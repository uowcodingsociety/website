Python
Collaboration between Warwick Coding Society, Computing Society and Warwick AI. This course assumes no prior knowledge and walks you through all of the programming fundamentals which are then applied to a fun project: Wahoot
6 Lessons - 12 hours total content
1. Setup, Variables, Data types, and Debugging
2. Operators, Conditional statements, and Functions
3. Data Structures, String Slicing, and Loops
4. File Handling & The Command line
5. Object Oriented Programming & Decorators
6. Efficiency, Compactness & Readability

7 Lessons - 14 hours total content
1. Basics of HTML
2. Basics of CSS
3. Project part 1: Setting up your workspace
4. Project part 2: Building & Styling the NavBar

Rust
This course provides an introduction to Rust, a modern systems programming language that prioritizes memory safety without sacrificing performance. Learn the fundamentals of ownership, borrowing, and concurrency while building practical projects in this beginner-friendly course.
3 Lessons - 6 hours total content
1. Introduction to Rust & Setup
2. Borrow Checker, Traits, & Smart Pointers
3. Tokio & the Rust Async Model
Swift
This course made by Adriano Barbet is an introduction to iOS dev. It goes over the basic syntax of Swift and the framework SwiftUI. It then walks through a project: Building the Instagram Home and Profile page UI.
6 Lessons - 12 hours total content
1. Basics of Swift & iOS Development
2. Basics of SwiftUI & Xcode
3. Project part 1: Setting up your work environment
4. Project part 2: Building the home page
5. Project part 3: Building the tab bar
6. Project part 4: Building the profile page
UI/UX
This course was made by Saffron Rabadiya. This course is an introduction to User Interface and User Experience design. It teaches you the fundamentals, and then walks you through a project, creating your portfolio website in figma.
6 Lessons - 12 hours total content
1. Intro to User Interface (UI) Design
2. Intro to User Experience (UX) Design
3. Intro to Prototyping
4. Intro to User Research
5. Prototyping Task
6. Portfolio Project in Figma
